Turn LinkedIn URLs into full contact profiles

Paste a list of LinkedIn profile URLs — Clay scrapes the profile, then FullEnrich enriches with verified email and phone number. Outreach-ready in minutes.

Get Template
1
Create your table
Click Get Template above to copy the Clay table into your workspace. Or create your own:
  • Click Import from CSV in Clay and upload a CSV with one column: LinkedIn URL
  • Or create a new table and add a basic text column called LinkedIn URL

Accepted URL formats:
LinkedIn URL
https://www.linkedin.com/in/johndoe
https://linkedin.com/in/janesmith
linkedin.com/in/alexjohnson
2
Add LinkedIn Profile enrichment
Add the "Enrich Person from LinkedIn" action column and map the input to your LinkedIn URL column. This extracts:
  • Full Name
  • Current Job Title
  • Company Name
  • Company Domain
  • Location / Country
  • Work experience history

Enrich Person from LinkedIn

0.5 credits / profile

Scrapes the LinkedIn profile to extract name, title, company, domain, location, and work history.

3
Add Work Email enrichment
Add the "FullEnrich Find Work Email" action column and map these inputs:
  • Full Name → from the "Enrich Person from LinkedIn" column
  • Company Domain → from the LinkedIn enrichment (current company domain)
  • LinkedIn URL → from your original LinkedIn URL column (for accuracy)

FullEnrich Find Work Email

1.5 credits / contact

Finds and verifies the person's professional work email address.

4
Add Mobile Phone enrichment
Add the "FullEnrich Find Mobile Phone" action column and map these inputs:
  • Full Name → from the "Enrich Person from LinkedIn" column
  • Company Domain → from the LinkedIn enrichment
  • LinkedIn URL → from your original LinkedIn URL column (for accuracy)

FullEnrich Find Mobile Phone

25 credits / contact

Finds the person's direct mobile phone number. Perfect for cold calling.

5
Run your enrichments
Paste your LinkedIn URLs into the first column (one per row), then run the enrichments in order:
  • First: Run the Enrich Person from LinkedIn column
  • Then: Run FullEnrich Find Work Email
  • Finally: Run FullEnrich Find Mobile Phone
You can run all rows at once or test with a few rows first.
6
Export your enriched list
All data sits in separate columns: name, title, company, email, phone. Export to:
  • CSV — for any tool
  • CRM — push directly to HubSpot, Salesforce, etc.
  • Email sequencer — Instantly, lemlist, Smartlead, etc.

Pro tips:
  • Test with 2–3 LinkedIn URLs first before running on your full list
  • Not all profiles will have findable emails/phones — success rates vary by data availability
  • You can set up conditional runs to only find phones for contacts where email was found

Troubleshooting

ProblemSolution
Clay returns no profile dataThe LinkedIn URL may be invalid or the profile is set to private. Double-check the URL format and try a public profile.
FullEnrich returns no emailMake sure Clay successfully scraped the name and company first — FullEnrich needs these fields to find the email.
Phone enrichment returns emptyMobile numbers are harder to find. Some regions have lower coverage. You can set up conditional runs to skip phone lookup when no email was found.
Enrichment stuck on "running"Wait a few minutes — large lists take time. If it stays stuck, try re-running the column on the affected rows.
