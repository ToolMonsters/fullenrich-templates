Tips for best results:
  • Use accurate domains — format as company.com (no www or https)
  • Run in batches — start with 10–20 companies to test before scaling
  • Check data quality — review first few results before running full list
  • Customize job titles — edit "Find Decision Makers" to target different roles if needed

Troubleshooting

ProblemSolution
No decision makers foundThe company may be too small or not well indexed. Try broader job titles or check the domain is correct.
Decision Makers table is emptyMake sure the "Send table data" action ran successfully. Check that Table 1 found results first.
FullEnrich returns no emailSome contacts don't have a discoverable work email. FullEnrich needs the name and company domain — make sure these were passed from Table 1.
Phone enrichment returns emptyMobile numbers are harder to find. Some regions have lower coverage. Consider running phone lookup only on contacts where email was found.
