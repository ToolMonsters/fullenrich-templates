Enrich new Typeform responses with FullEnrich

Automatically enrich every new Typeform submission with verified B2B emails, phone numbers, and LinkedIn profiles using FullEnrich, then push the enriched data to Google Sheets.

What this Zap does: When someone submits your Typeform, FullEnrich looks up the respondent's verified B2B email and phone number using their name and company, then adds the enriched data to a Google Sheets spreadsheet.
1
Create a new Zap
Go to zapier.com and click Create Zap. For the trigger, search for Typeform and select New Entry. Connect your Typeform account and select the form you want to use. Make sure your form collects First Name, Last Name, and Company.
2
Test the trigger
Click Test trigger. Zapier will pull a recent Typeform response. Make sure the First Name, Last Name, and Company fields appear in the test data.
3
Add FullEnrich
Add a new action step and search for FullEnrich. Select the action Enrich Contact. Connect your FullEnrich account using your API key. Then map the fields:
  • First Name → Typeform's First Name answer
  • Last Name → Typeform's Last Name answer
  • Company Name → Typeform's Company answer
  • Company Domain → Typeform's Website answer (if collected)
4
Add a row to Google Sheets
Add another action step: Google Sheets → Create Spreadsheet Row. Select your spreadsheet and worksheet. Map the columns:
  • First Name → Typeform answer
  • Last Name → Typeform answer
  • Company → Typeform answer
  • B2B Email → Most Probable Email (from FullEnrich)
  • Phone → Most Probable Phone (from FullEnrich)
5
Test and publish
Test the full Zap, then click Publish. Every new Typeform submission will now be automatically enriched with verified B2B contact data and logged in Google Sheets.

Troubleshooting

ProblemSolution
FullEnrich returns no emailMake sure First Name, Last Name, and Company are mapped correctly. Adding the company website/domain improves match rates.
Wrong Typeform fields appearEnsure your form has separate questions for First Name, Last Name, and Company. A single "Full Name" field won't map correctly.
Google Sheets row is incompleteCheck that the spreadsheet columns match the mapped fields. Create headers in the first row before testing.
What this scenario does: When someone submits your Typeform, FullEnrich looks up the respondent's verified B2B email and phone number, then adds the enriched data to a Google Sheets spreadsheet.
1
Create a new scenario
Go to make.com and click Create a new scenario. Add the first module: search for Typeform and select Watch Responses. Connect your Typeform account and select the form you want to watch.
2
Add FullEnrich
Click the + to add a new module. Search for FullEnrich and select Enrich a Contact. Connect your FullEnrich API key. Map the fields:
  • First Name → Typeform's First Name answer
  • Last Name → Typeform's Last Name answer
  • Company Name → Typeform's Company answer
  • Company Domain → Typeform's Website answer (if collected)
3
Add a row to Google Sheets
Add a third module: Google Sheets → Add a Row. Connect your Google account, select the spreadsheet and worksheet. Map the columns:
  • First Name → Typeform answer
  • Last Name → Typeform answer
  • Company → Typeform answer
  • B2B Email → Most Probable Email (from FullEnrich)
  • Phone → Most Probable Phone (from FullEnrich)
4
Test and activate
Click Run once to test. Submit your Typeform and check that a new row appears in Google Sheets with the enriched data. Once confirmed, toggle the scenario ON.

What this workflow does: When someone submits your Typeform, FullEnrich enriches the respondent with verified B2B email, phone, and LinkedIn URL, then adds the enriched data to Google Sheets. This workflow uses two parallel flows because FullEnrich works asynchronously — it sends results to a webhook when the enrichment is ready.
Why two flows? FullEnrich enrichment takes a few seconds. Instead of waiting, FullEnrich sends the results to a webhook when ready. The top flow sends the request; the bottom flow receives and processes the results.
1
Create the top flow: Typeform trigger
Create a new workflow in n8n. Add a Typeform Trigger node. Connect your Typeform account and select the form you want to watch. This node fires instantly when a new response is submitted.
2
Add the FullEnrich node
Add a FullEnrich node after the Typeform trigger. Connect your API key and map First Name, Last Name, Company Name, and Company Domain from the Typeform answers. Set the Webhook URL to the webhook node's production URL (from Step 3). Add a Custom Field to pass through a unique identifier (e.g. the Typeform response ID).
3
Create the bottom flow: Webhook
Add a Webhook node (this starts a separate flow). Set the HTTP method to POST and give it a path like typeform-enrich. Copy the production URL and paste it in the FullEnrich node's Webhook URL field.
4
Add a row to Google Sheets
After the Webhook node, add a Google Sheets node set to Append Row. Connect your Google account and select the target spreadsheet. Map the columns from the FullEnrich response:
  • B2B Email → most_probable_email
  • Phone → most_probable_phone
  • LinkedIn → social_medias[0].url
5
Test and activate
Click Test workflow, then submit a response on your Typeform. After a few seconds, the webhook receives the enriched results and a new row appears in Google Sheets. Toggle the workflow to Active and make sure the Webhook URL is set to the production URL.

Troubleshooting

ProblemSolution
Webhook never receives dataMake sure you're using the production webhook URL (without webhook-test). Also make sure the workflow is Active.
FullEnrich returns no emailMake sure First Name, Last Name, and Company are mapped correctly. Adding Company Domain improves match rates.
Google Sheets row is incompleteCheck that the spreadsheet columns match the mapped fields. Create headers in the first row before testing.
